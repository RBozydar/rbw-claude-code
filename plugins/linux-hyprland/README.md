# Linux-Hyprland

Skills for building custom desktop shells on Hyprland using Quickshell (Qt6/QML).

## Skills

### quickshell

Comprehensive Quickshell development skill covering:

- **SKILL.md** — Core architecture, Hyprland integration, IPC wiring, tooling (qmlls/qmlformat/qmllint), 27 gotchas, verification steps
- **references/qml-patterns.md** — 32 Quickshell-specific code patterns (bars, singletons, audio, lock screens, IPC, config persistence, etc.)
- **references/modules-api.md** — Full API reference for all 18 Quickshell QML modules
- **references/visual-effects.md** — MultiEffect (blur, shadow, mask, colorize), ShaderEffect, layer system, Canvas, compositor-level blur
- **references/qt-essentials.md** — Core Qt/QML: focus, scrolling, states/transitions, colors, gradients, animations, utility functions
- **references/best-practices.md** — 10 architecture patterns: design tokens, theming, error handling, logging, performance, state management
- **references/feature-matrix.md** — Feature-by-feature comparison across 4 reference shells with exact file paths
- **references/reference-repos.md** — Indexed map of local reference repositories
